# GPX-Track-Visualizer
# How to Use:
Only the first time:
```
chmod +x compile\ plot.cpp.sh && ./compile\ plot.cpp.sh
```
Run:
```
./gpx_visualizer /path/to/.gpx/files
```
# Hotkeys:
```
P:       Lines / Points
S:       Speed
Up/Down: Change line distance threshold
Scroll:  Zoom
LMB:     Move
```
# Color
```
Dark blue:   0km/h
Light Blue:  5km/h
Green:      10km/h
Yellow:     20km/h
Orange:     30km/h
Red:        50km/h
```
